# Formula 1 Race Session Intelligence and Telemetry Visualization

This application, built using Next.js, serves as a powerful tool for visualizing telemetry data from Formula 1 races. It integrates advanced features for an enhanced user experience, including:

- **Offline Demo Playback**: Users can access previously recorded race sessions at any time, allowing them to analyze telemetry and race dynamics without the need for a live connection.
- **Real-Time AI-Powered Commentary**: The application utilizes the Airia API to provide dynamic commentary during races, delivering insights and enhancing viewer engagement.

Whether you're a dedicated fan or a data enthusiast, this app offers a comprehensive platform to explore and understand the intricacies of Formula 1 racing.
